## Download Instructions
Method 1 (Recommended):
1. Download GitKraken for free from https://www.gitkraken.com/
2. Open GitKraken and click "Clone with URL" and pase https://github.com/Tavin-H/MATH232-Notes/ (make sure to remember where you're cloning to!)
3. Download Obsidian from https://obsidian.md/
4. Open obsidian and click "Open" next to "Open folder as Vault"
5. Navigate to where you cloned the repo and open that folder
6. To download the most recent changes, go to GitKraken and click "Pull" if you've made any changes to the file this may not work
7. Have fun studying!

Method 2:
1. Download Obsidian from https://obsidian.md/
2. In this repo home page click "code" and then "Download as ZIP"
3. Unzip the folder
4. Open obsidian and click "Open" next to "Open folder as Vault"
5. Navigate to where your unzipped folder is and open it.
6. Happy studying!

The reason this method is not recommended is because to update the notes when I add things, you will need to redownload, unzip, and reopen the folder each time
